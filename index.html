🏠 系统总览

抖音看板是一套多账号视频数据管理系统，支持管理员和员工双角色，数据实时同步至云端。

🖥️ 界面结构
左侧边栏 — 账号列表、文件夹切换、导出工具
主内容区 — 视频卡片展示（宫格/缩略/列表三种视图）
顶部栏 — 搜索、筛选、视图切换、添加视频
👤 双角色说明
管理员 拥有全部功能：增删账号/视频/用户，数据分析，CSV 导出
员工 仅能查看所属文件夹内的账号和视频，可搜索和上传视频
💾 数据说明
所有数据存储在 Supabase 云端数据库，多设备实时同步。点击顶部 ↺ 刷新 按钮可手动拉取最新数据。

👤 账号管理

1
添加账号：点击左侧边栏底部「＋ 添加账号」，填写账号名称、颜色标签、运营备注，可选绑定文件夹。
2
编辑账号：点击账号列表中任意账号的 ✏️ 编辑 按钮，可修改名称、颜色、运营周期等信息。
3
删除账号：点击 🗑 删除 按钮，该账号下所有视频数据将同步从数据库删除，操作不可逆，请谨慎。
4
文件夹分组：点击「＋ 添加文件夹」创建分组，在账号编辑中将账号绑定至指定文件夹，便于员工权限分配。
📊 账号统计栏
侧边栏顶部实时展示：总视频数 📹、违规数 ⚠、总点赞 ❤、推广数 📢。

🎬 视频管理

1
添加视频：点击右上角「＋ 添加视频」，粘贴抖音视频链接，系统会自动解析封面和标题。
2
填写数据：输入播放量、点赞、评论、收藏、转发数，选择发布日期，勾选是否违规或推广。
3
编辑视频：点击视频卡片右上角 ✏ 按钮，可修改所有数据字段。保存后实时更新云端。
4
删除视频：点击视频卡片 🗑 按钮，确认后从数据库永久删除。
🔍 搜索与筛选
顶部搜索框：按视频标题实时过滤
筛选按钮：全部 ⚠ 违规 📢 推广 ✓ 正常
视图切换：⊞ 宫格 / ⊡ 缩略 / ☰ 列表
📥 导出 CSV
点击侧边栏底部「📥 导出全部数据 CSV」，导出所有账号下全部视频数据为 Excel 可读格式。

📊 数据分析

点击账号列表中任意账号的 📊 数据分析 按钮，进入该账号的深度分析面板。

📈 图表分析
切换播放/点赞/评论/收藏/转发/时长图表，直观查看各数据项的趋势和分布。
🏥 到诊分析
录入周到诊或月到诊数据后，系统自动计算：
· 每万播放的到诊转化率
· 各互动指标与到诊的关联分析
· 达到目标到诊量所需的各项指标预测
📅 周期分析 / 深度分析
基于多周/多月历史数据，输出综合转化率、效率评级、瓶颈识别和优先优化方向建议。

👥 用户权限管理

管理员可通过侧边栏底部「👥 用户管理」按钮管理系统用户。

1
添加员工：填写用户名、密码，选择绑定文件夹，设置角色为「员工」。
2
文件夹绑定：员工只能查看其绑定文件夹内的账号和视频，实现数据隔离。
3
删除员工：删除员工账号后，该员工上传的视频「上传者」标签自动清空，视频数据保留。
🔒 员工视图特性
员工登录后进入专属视图，只能看到所属文件夹内容。
支持：视频搜索、视频上传标注、切换账号查看数据分析。
不支持：账号管理、文件夹管理、CSV导出、用户管理。

💡 使用技巧

🌙 深色模式
点击左上角 🌙 按钮切换深色/浅色模式，偏好设置自动保存，下次打开自动恢复。
🔄 数据刷新
顶部栏 ↺ 按钮手动刷新当前账号数据。建议在多人协作录入数据后手动刷新以获取最新状态。
📌 切换账号/文件夹时的注意事项
切换账号或文件夹时，搜索框会自动清空，避免旧搜索词干扰新内容的展示。
📊 图表最佳实践
建议在视频数量 ≥ 10 条后再查看趋势分析，数据量越大分析结果越准确。到诊分析需先录入至少一条周/月到诊记录。
🚪 多设备使用
系统支持多设备同时在线，数据云端同步。登录信息保存在浏览器 sessionStorage，关闭标签页后需重新登录。
💬 Chatbox 文案助手
点击侧边栏（或顶部栏）的 💬 Chatbox 文案助手 按钮，可在新标签页打开 AI 文案工具。
首次使用需在设置中配置 API：
① 点击右上角设置 → 模型提供商 → 添加自定义
② API Mode 选 OpenAI API Compatible
③ 填入团队提供的 API Key 和 API Host
④ 点击 Fetch 获取模型列表，选择模型后即可使用
适用场景：视频文案撰写、标题优化、评论回复、推广话术生成等。
