🏠 系统总览

抖音看板是一套多项目视频数据管理系统，支持管理员和员工双角色，数据实时同步至云端，配合浏览器插件可一键抓取抖音数据。

🖥️ 界面结构
左侧边栏 — 图标导航，点击可展开项目/账号列表、视频排名、导出工具
主内容区 — 视频卡片展示，支持宫格 / 列表 / 排名 / 看板多种视图
顶部栏 — 搜索、状态筛选、视图切换、刷新、添加视频
👤 使用角色
管理员 拥有全部功能：增删项目/视频，数据分析，CSV 导出，视频排名
多用户权限系统即将上线，敬请期待
💾 数据说明
所有数据存储在 Supabase 云端数据库，多设备实时同步。点击顶部 ↺ 刷新 按钮可手动拉取最新数据。
✨ 最新功能
· 评论三要素扫描：插件自动扫描评论区，检测关键词是否带齐，结果显示在视频卡片上
· 视频排名：按点赞/播放/评论等指标查看 TOP 50 视频排名
· 批量三要素扫描：批量更新数据时同步扫描所有视频的评论三要素
· 点卡片直接编辑：点击视频卡片任意位置即可打开完整编辑面板

📁 项目管理

1
添加项目：点击侧边栏「项目 / 账号」→「＋ 添加账号」，填写项目名称、颜色标签、运营备注，可选绑定文件夹。
2
编辑项目：在数据总览页面的账号列表中，点击账号旁的编辑按钮，可修改名称、颜色、运营周期等信息。
3
文件夹分组：点击「项目 / 账号」菜单里的「＋ 新建文件夹」，然后在项目编辑中绑定文件夹，用于员工权限隔离。
4
删除项目：删除后该项目下所有视频数据同步从数据库删除，操作不可逆，请谨慎确认。
📊 项目统计
数据总览页面实时展示各项目的视频总数、总点赞、违规数和推广数，点击项目可查看深度分析。

🎬 视频管理

1
添加视频：点击右上角「＋ 添加」，粘贴抖音视频链接，填写播放/点赞/评论/收藏/转发等数据，选择归属项目和发布日期。
2
编辑视频：点击视频卡片任意位置即可打开完整编辑面板，修改所有字段后保存实时同步云端。
3
删除视频：点击视频卡片右下角 🗑 按钮，确认后从数据库永久删除。
4
批量操作：点击顶部多选按钮进入选择模式，勾选多个视频后可批量移动项目或批量删除。
🔍 搜索与筛选
顶部搜索框：按视频标题实时过滤
筛选按钮：全部 ⚠ 违规 📢 推广 ✓ 正常
视图切换：⊞ 宫格 / ☰ 列表
🏆 视频排名
点击侧边栏「视频排名」图标，可查看按点赞/播放/评论/转发/收藏排序的 TOP 50 视频，点击跳转对应项目。
💬 三要素徽章
通过插件扫描过评论的视频，卡片底部会显示 ✓ 三要素(3/3) 或 ✗ 三要素(1/3)，直观反映评论带齐情况。
📥 导出 CSV
点击侧边栏「导出 CSV」图标，导出所有项目下全部视频数据为 Excel 可读格式。

🧩 浏览器插件使用

配合专属浏览器插件，可直接从抖音页面抓取数据、扫描评论三要素，无需手动填写。

📱 插件主界面 — 保存视频 + 评论扫描
1 在抖音视频页面点击插件图标，点「读取数据 + 扫描评论」
2 插件自动读取播放/点赞等数据，同时打开评论区扫描关键词
3 扫描完成后显示三要素是否带齐（✓ / ✗）及各关键词命中数
4 点「保存到看板」，视频数据和扫描结果同时写入数据库
5 回看板刷新，视频卡片上即显示三要素徽章
⚙️ 关键词配置
在插件「关键词」区域添加要检测的词（空格分隔可批量添加）
在「三要素（勾选3个）」中选择最重要的3个词作为必须带齐的要素
设置「最多扫描评论数」（默认200条）
配置会自动保存，批量更新时也会读取同一套配置
🔍 关键词匹配规则
支持四种匹配方式，任一命中即算：
· 精确匹配：评论包含完整关键词
· 去空格匹配：「公 众 号」→「公众号」
· 模糊包含：评论含关键词所有字即命中（「贵州银屑病白癜风皮肤医院」→ 命中「贵州银屑病医院」）
· 拼音首字母：「gzh」→ 命中「公众号」「共中号」等谐音写法
🔄 批量更新 + 批量扫描
1 在插件中点击「批量更新」打开批量更新页面
2 选择要更新的文件夹和日期范围
3 开启「同时扫描评论三要素」开关（需先在主界面配置关键词）
4 点击「开始批量更新」，插件逐条访问视频页面，自动刷新数据并扫描评论
5 每条视频完成后显示更新状态和三要素结果（✓ 3/3 或 ✗ 1/3）

注意：批量更新期间请勿关闭浏览器，等待全部完成后再操作。

📊 数据分析

点击数据总览页面中任意项目的分析区域，进入该项目的深度分析面板。

📈 图表分析
切换播放 / 点赞 / 评论 / 收藏 / 转发图表，直观查看各数据项的趋势和分布。视频数量 ≥ 10 条时分析结果更准确。
🏥 线索分析
录入周线索或月线索数据后，系统自动计算：
· 每万播放的线索转化率
· 各互动指标与线索的关联分析
· 达到目标线索量所需的各项指标预测
📅 周期分析 / 深度分析
基于多周 / 多月历史数据，输出综合转化率、效率评级、瓶颈识别和优先优化方向建议。
🏆 全局视频排名
侧边栏「视频排名」功能可跨项目查看 TOP 50 视频，支持按点赞 / 播放 / 评论 / 转发 / 收藏五个维度切换。

💡 使用技巧

🎨 主题配色
点击侧边栏设置 → 「主题配色」，可切换暖橙 / 深海蓝 / 薄荷绿 / 玫瑰红 / 紫罗兰 / 极夜深色六套主题，偏好自动保存。
🌙 深色模式
点击设置菜单中「深色 / 浅色模式」切换，或直接点击侧边栏月亮图标，偏好设置自动保存。
🔄 数据刷新
顶部栏 ↺ 按钮手动刷新数据。多人协作时，建议他人录入后手动刷新获取最新状态。插件保存后也需刷新才能看到三要素徽章。
📌 搜索与筛选
切换项目或文件夹时，搜索框自动清空，避免旧搜索词干扰新内容展示。筛选状态（违规/推广/正常）切换项目后同样重置。
🚪 多设备使用
系统支持多设备同时在线，数据云端同步。登录信息保存在浏览器 sessionStorage，关闭标签页后需重新登录。
💬 Chatbox 文案助手
点击设置菜单中的「Chatbox 文案助手」链接，可在新标签页打开 AI 文案工具。
首次使用需在 Chatbox 设置中配置 API Key，联系管理员获取团队 API Key 和 Host 地址。
适用场景：视频文案撰写、标题优化、评论回复、推广话术生成等。
📖 员工使用教程
抖音看板员工版 · 快速上手指南
🏠 员工版界面说明
员工登录后进入专属视图，只能查看所属文件夹内的项目和视频数据。
顶部导航栏 — 医枢云库、Chatbox文案助手、添加视频、搜索、视图切换、刷新、教程、退出
主内容区 — 视频卡片展示，支持宫格 ⊞ / 列表 ☰ / 紧凑 ⊡ 三种视图
🎬 如何添加视频
1
点击顶部「＋ 添加视频」按钮，粘贴抖音视频链接。
2
填写点赞、评论、收藏、转发数，选择发布日期，勾选是否违规或推广。
3
点击「保存」，视频数据自动同步至云端，所有人可见。
✏️ 编辑 / 删除视频
点击视频卡片右上角 ✏ 按钮可修改数据，保存后实时更新云端。
点击 🗑 按钮删除视频，操作不可逆，请谨慎确认。
🔍 搜索视频
顶部搜索框输入关键词，实时过滤当前项目下的视频标题。
点击 ↺ 刷新按钮，手动拉取最新数据（多人协作时建议刷新）。
📊 查看数据分析
在项目列表页点击项目名称旁的「📊 数据分析」按钮，可查看该项目的播放趋势、点赞分布、线索转化等图表。
💬 Chatbox 文案助手使用方法
点击顶部「💬 Chatbox 文案助手」按钮，新标签页打开 AI 文案工具。
首次使用需配置 API：
① 点击右上角设置 → 模型提供商 → 添加自定义
② API Mode 选 OpenAI API Compatible
③ 填入团队提供的 API Key 和 API Host
④ 点击 Fetch 获取模型列表，选择模型后即可使用
适用：视频文案撰写、标题优化、评论回复、推广话术生成等。
🧩 浏览器插件 · 一键导入抖音数据
配合浏览器插件，可直接从抖音页面抓取视频数据导入看板，无需手动填写数据。

使用步骤：
① 在浏览器打开抖音，搜索话题或刷视频，进入目标视频页面
② 点击浏览器右上角的插件图标，插件自动读取当前页面数据
③ 插件同时截图当前画面作为视频封面
④ 点赞、收藏、评论、转发数自动填入，确认后一键导入数据库
⑤ 回到看板，数据即刻在视频卡片上显示
🔄 批量更新功能
对已录入看板的视频，可使用插件批量刷新所有数据，无需逐条手动编辑。

使用步骤：
① 在看板中选择需要更新的项目或文件夹
② 打开插件，插件识别看板中已有视频列表
③ 点击插件中的「批量更新」按钮
④ 插件自动逐条抓取最新点赞、收藏、评论、转发数
⑤ 抓取完成后数据同步写入数据库，看板自动更新

注意：批量更新期间请勿关闭浏览器或插件窗口，等完成提示后再操作。
💡 注意事项
· 员工只能查看所属文件夹内的内容，无法查看其他文件夹数据
· 不支持项目管理、文件夹管理、CSV 导出、用户管理
· 登录信息保存在浏览器 sessionStorage，关闭标签页后需重新登录
· 如遇数据异常，请点击 ↺ 刷新或联系管理员
